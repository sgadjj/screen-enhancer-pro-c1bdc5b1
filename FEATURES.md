# Game Enhancer Pro - مميزات التطبيق

## iPad Gaming Mode
Optimized layout and controls for landscape iPad gaming experience with larger text and better spacing.

## Filter Controls
- **Contrast**: Adjust screen contrast (50-200%)
- **Brightness**: Control overall brightness (50-200%)
- **Saturation**: Modify color saturation (0-200%)
- **Sharpness**: Enhance image sharpness (0-100%)
- **Background Flattening**: Reduce background saturation and contrast for better foreground visibility

## Object-Specific Adjustments
Fine-tune visual settings separately for:
- **Moving Objects**: Adjust brightness (50-200%) and color intensity (0-100%) for moving targets
- **Stationary Objects**: Separate controls for background elements

## Color Enhancement
- **Color Hue Selector**: Choose specific color tones to highlight (0-360°)
- **Color Intensity**: Adjust the strength of color highlighting

## Draggable Target Icon
- **Animated Crosshair**: Shows a pulsing target icon overlay
- **Fully Draggable**: Touch and drag to position anywhere on screen
- **Hide/Show Toggle**: Easily toggle visibility with the "Target Icon" button
- **Hover Controls**: Shows move handle and close button on hover/touch
- **Gaming Aid**: Use as a reference point for aiming practice

## Preset Management
- **Save Custom Presets**: Store your favorite filter combinations
- **Quick Load**: Apply saved presets with a single tap
- **Delete Presets**: Remove presets you no longer need

## System Integration
- **Shizuku Integration**: Uses Shizuku for system-wide overlay permissions
- **Overlay Permission Management**: Automatic permission handling for Android
- **Floating Toggle**: Quick on/off button that stays accessible

## Technical Features
- Real-time filter application
- Hardware-accelerated rendering
- Minimal battery impact
- Works over games and other apps (with proper permissions)
- Responsive design optimized for tablets and phones

## How to Use in Games
1. Enable Shizuku and grant overlay permissions
2. Configure your desired filters and object-specific settings
3. Enable the Target Icon if needed for aiming reference
4. Toggle filters on/off with the floating button
5. The filters apply over your game in real-time
